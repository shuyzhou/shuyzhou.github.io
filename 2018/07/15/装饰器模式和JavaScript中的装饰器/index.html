装饰器模式和JavaScript中的装饰器

2018-07-15 JavaScript

装饰器模式可以在代码执行时，给某个对象动态地添加或修改某个功能，而不改变其本身的结构。

装饰器和继承都可以实现对对象的修改，但装饰器明显更简单灵活，并且不会影响到对象的父类或子类。
