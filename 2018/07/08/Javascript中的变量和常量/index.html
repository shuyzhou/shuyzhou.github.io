2018-07-08 Javascript

变量、常量声明方式

在Javascript中，声明常量的方式只有一种：const，而声明变量的方式有两种：var和let。

被var声明的变量，在同一作用域内可以被var再次声明并赋值，它的值将为最新的值。而被let和const声明的变量和常量不允许与同一作用域内其他变量、常量或函数名相同。

以var和let为例看下两者的区别：由于var声明的其实是全局变量，因此console.log函数在执行时会在全局作用域对i进行标识符解析，而这时i的值已经变为3；而在for循环里let声明的变量i在每次进入循环体时就产生一个块级作用域，并且把i当时的值绑定在该作用域里，console.log函数在执行时会对相应的块级作用域做标识符解析，所以会打印出不同的值。

变量提升

var声明的变量以及函数会经过变量提升，即不管在何处声明变量都等同于在函数的最开始声明该变量。Javascript引擎会在函数代码执行前处理所有var声明的变量声明，并将undefined赋值给变量：

同样，函数声明也会经过变量提升，但会被限制在声明它的语句块内。所以，最好还是将变量声明放在函数的顶部，以免出现TypeError异常。
