ES6中的Proxy

2018-07-08 Javascript

Proxy是ES6中定义的一个浏览器内置构造函数，利用它可以对一些操作符和Object的方法进行重载，实现对数据访问的监听。Proxy构造函数接受两个参数：target和handler。target是需要代理的对象（它可以为任意一个对象）；handler为一些函数（这些函数被称为trap）的集合，当对代理对象进行某些操作时，这些操作将转发到对应的trap里。Proxy基本语法如下：

traps对应操作

trap 对应操作
getPrototypeOf Object.getPrototypeOf
setPrototypeOf Object.setPrototypeOf
isExtensible Object.isExtensible
preventExtensions Object.preventExtensions
getOwnPropertyDescriptor Object.getOwnPropertyDescriptor
defineProperty Object.defineProperty
has in操作符
get 获取对象某属性值
set 给对象某属性赋值
deleteProperty delete操作符
ownKeys Object.getOwnPropertyNames 和 Object.getOwnPropertySymbols
apply 调用函数
construct new操作符
