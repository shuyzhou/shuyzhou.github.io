HTML语义化标签（三）

2017-03-30 HTML

插入文本

<ins>标签定义已经被插入文档中的文本。

图像映射

<map>定义一个带有可点击区域的图像映射，<area>定义图像中可点击区域和其映射的url，img元素中的 “usemap” 属性引用 map 元素中的 “id” 或 “name” 属性。

标记文本

<mark>标签定义带有记号的文本。

导航栏

<nav>标签定义导航链接的部分。

进度

<progress>标签标示任务的进度（进程）

视频

<video>标签定义视频，比如电影片段或其他视频流。
