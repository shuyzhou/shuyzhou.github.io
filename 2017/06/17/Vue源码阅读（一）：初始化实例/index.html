Vue源码阅读（一）：初始化实例

2017-06-17

###

function Vue (options) {
this._init(options)
}
// install internals
initMixin(Vue)
stateMixin(Vue)
eventsMixin(Vue)
lifecycleMixin(Vue)
miscMixin(Vue)
// install instance APIs
dataAPI(Vue)
domAPI(Vue)
eventsAPI(Vue)
lifecycleAPI(Vue)

//…实例属性的初始化操作
// push self into parent / transclusion host
if (this.$parent) {
this.$parent.$children.push(this)
}
options = this.$options = mergeOptions(
this.constructor.options,
options,
this
)

// call init hook
this._callHook(‘init’)
// initialize data observation and scope inheritance.
this._initState()
// setup event system and option events.
this._initEvents()
// call created hook
this._callHook(‘created’)
// if el option is passed, start compilation.
if (options.el) {
this.$mount(options.el)
}
`` _init函数建立父子实例之间的联系，合并了自定义option和默认option，调用相应的生命周期的回调函数，初始化data observation机制和Vue事件机制，如果确定了Vue实例的作用域的话还要挂载Vue实例。可以看出，生命周期是独立的一部分，主要操作集中于_initState、_initEvents和$mount三个函数。在下一篇博客里，我将分析_initState中的_initData函数，以便理解Vue`是怎样监听数据变化的。
